# AI-Ransomware-Detection usig Deep COntracrive Autoencoder
Developed a deep learning based anomaly detection system for identifying ransomware behavior. The model uses a Contractive Autoencoder trained on benign samples and detects malicious behavior using reconstruction error thresholds.
The model was trained on a balanced malware dataset containing 21,752 PE samples with 76 static features. A Contractive Autoencoder was trained only on benign samples and used reconstruction error for anomaly detection.
